Strict scheme: lowercase = soft (h → ሀ), uppercase = hard / emphatic (H → ሐ, T → ጠ, P → ጰ, Ch → ጨ, Ts → ፀ). Vowels: e u i a ie o (1st–5th, 7th orders); bare consonant = 6th order (b → ብ). ' for the ayn series ('a → ዓ), I for standalone እ, W for labialized rows (kWa → ኳ).
